# 🏨 The Flame Tree Hotel ERP System

The Flame Tree Hotel ERP is a comprehensive, enterprise-grade Information System designed to automate core hospitality, logistics, and supply chain operations. Replicating real-world ERP software behavior (like SAP/Odoo), the system bridges the gap between daily hotel workflows and strict enterprise control.

### 🚀 Key Business Capabilities Covered:
* **Material Management & Procurement (SAP MM Logic):** End-to-end automation of the Procure-to-Pay (P2P) lifecycle, including vendor catalogs, purchase order tracking, and dynamic inventory level reconciliation.
* **Workflow Automation & Approval Matrices:** A role-based notification and approval workflow allowing Supervisors and Managers to digitally authorize high-value procurement requests.
* **Operational Control:** Centralized modules for Guest Services, Plant Maintenance (Damaged Item Tracking), and Housekeeping inventory usage auditing.

## 🎯 Core Focus: Procurement & Supplier Management (My Contribution)

As my primary focus in this enterprise application, I designed and developed the complete end-to-end Supply Chain and Procurement logic, replicating core enterprise workflows (similar to SAP Material Management module behavior).

### Key Functional Capabilities Implemented:
* **Automated Procure-to-Pay (P2P) Lifecycle:** Developed the system architecture to handle the seamless progression from Purchase Requisition creation to final Purchase Order approval.
* **Supplier & Vendor Management Directory:** Built a structured database and user interface to onboard suppliers, track active vendors, and log product catalogs.
* **Granular Role-Based Approval Matrix:** Implemented backend validation layers ensuring procurement workflows require digital verification and multi-level authorization from higher-level management before execution.
* **Inventory Reconciliation Logic:** Connected the received goods status directly to the central ledger to automate real-time stock adjustments upon stock review confirmation.
